# To-Do-App

Taski – Dockerized Flask To-Do Application

Taski is a lightweight, production-style To-Do web application built with Python Flask and fully containerized using Docker. The project demonstrates the complete lifecycle of modern application containerization, including multi-stage Docker builds, Docker Compose orchestration, and secure deployment practices.

The application allows users to manage tasks efficiently with features such as adding tasks with priority levels, marking tasks as complete, deleting tasks, filtering by status, and viewing real-time task statistics.

This project is developed as a DevOps/containerization lab implementation, focusing on best practices such as image optimization, non-root container execution, health checks, and environment configuration using Docker Compose.

Key Features:

	Add, update, and delete tasks
	Priority-based task management (Low / Medium / High)
	Task filtering (All / Active / Completed)
	Live task statistics
	Simple Flask-based web interface
	Fully containerized using Docker
	Multi-stage Docker build for optimized image size
	Docker Compose orchestration

Tech Stack:

	Python (Flask)
	HTML, CSS
	Docker
	Docker Compose

Learning Outcomes:

	Containerizing a web application using Docker
	Writing multi-stage Dockerfiles
	Using Docker Compose for service orchestration
	Applying container security best practices
	Understanding deployment workflow in modern DevOps
